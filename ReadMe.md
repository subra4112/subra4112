### 🔬 Major Projects

🧠 **[Bitcoin Price Forecasting](https://github.com/subra4112/bitcoin-price-prediction)**  
Predictive modeling using SARIMAX, LSTM, and Prophet for 10 years of BTC data. Achieved 75–85% accuracy using RMSE, MAPE.

🌐 **[LightRAG vs Traditional QA with Neo4j](https://github.com/subra4112/graph-based-qa-lightRAG)**  
Built a knowledge graph of 1,700+ entities for context-aware Q&A using local LLMs (LLaMA 3.2), visualized via Streamlit dashboard.

📱 **[Husky Smart Mobility App](https://github.com/subra4112/autonomous-vehicle-app)**  
Designed a mobile app for booking AV transport, with 18% improved scheduling efficiency using Python & SQL.
